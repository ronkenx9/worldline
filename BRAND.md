# WORLDLINE — Brand Lock

> Locked decisions only. The exhaustive exploration lives elsewhere; this is what we build to.

## Core DNA

| | |
|---|---|
| **Name** | WORLDLINE (wordmark: all-caps `WORLDLINE`) |
| **Hero line** | Every world remembers. |
| **Tagline** | One player. Many worlds. One continuous story. |
| **Category** | Cross-game memory layer for AI worlds |
| **Enemy** | Game amnesia — every world forgets you the moment you leave. |
| **The loop** | **Act → Remember → Reconcile → React** (matches the real code) |
| **Pitch close** | The player is the platform. |

**What it is, in one sentence:** WORLDLINE turns a player's actions in one game into
portable, verified *canon* that other games can read and react to.

## Lexicon (these are the ONLY proper nouns — they're already the code's names)

- **Soul** — the player's persistent, self-sovereign identity/memory (event-sourced).
- **Canon** — the bounded shared world-state.
- **Continuity** — the agent that reconciles events into canon (reasons, not last-write-wins).
- **Gamemaster** — a per-game agent that authors its game's experience.
- **Keeper** — the privacy/guardian agent (Seal-encrypted soul attributes).
- **Adapter** — how a game plugs into the layer (the SDK seam).

No other brand names. (No "TrueSoul" sub-brand, no protocol/product split — one name.)
"Soul" carries the emotional hook; everything else is component nomenclature.

## Decisions resolved

- **Single brand**, not a two-tier product/protocol stack. One word on the slide.
- **Standalone** identity — NOT sub-branded as Kaizenverse. Packages must not use `kaizenverse`.
- **Aesthetic: game-native, luminous, mythic-infrastructure.** A library of living souls,
  not an enterprise dashboard, not generic web3 (no coins/cubes/nodes/gamepads/robots).
  Landing + assets should be **creative-forward and bold** — this is a *game* layer.
- **Trademark note:** "Worldline" is also a large public payments company (Worldline SA).
  Fine for the hackathon; if we keep the name long-term, revisit (or use the "Wordline" respelling).

events as a constellation; canon as a small gold-sealed archive object.

exits another. Thin rings, single line, one small glow node. No literal globe/gamepad/robot.

## Demo story (already proven in `src/demo/fullstack-run.ts`)

Betray one world → be remembered in the next.
Nightfall: steal the Eastern relic → Continuity marks `oathbreaker` → Freeports: guards
refuse entry. Inspector: same Soul, two games, one Canon — durable on Walrus + Sui + MemWal.
